# Privacy Policy for Readmd

Effective date: 2026-02-25

Readmd respects your privacy.

## What data we collect
Readmd does not collect, store, or transmit personal data to external servers.

## What the extension stores locally
Readmd stores only user preference settings using Chrome storage:
- Theme selection
- Layout selection (single-page or dual-page)
- Font size

These settings are used only to restore your reading preferences.

## Website content handling
When you activate reading mode, Readmd processes page content locally in your browser to render a cleaner reading view.
No article content is sent to remote servers.

## Permissions usage
- `storage`: Save user preferences.
- `activeTab`: Access the currently active tab when the user triggers the extension.
- `scripting`: Inject reading mode scripts/styles into the active page on user action.

## Third-party code
Readmd includes Mozilla Readability.js for content extraction. It runs locally in the browser context.

## Data sharing
Readmd does not sell, share, or transfer user data to third parties.
